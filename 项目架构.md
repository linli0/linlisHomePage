## 技术栈

### 后端 (Spring Boot 3.2 + Java 17)
- **框架**: Spring Boot 3.2, Spring Security, JJWT 0.12.3
- **数据库**: H2 (开发) / MySQL 8 (生产)
- **构建**: Maven 3.8+
- **认证**: JWT Bearer Token (密码-only 模式)
- **AI 集成**: Ollama 本地 LLM 代理

### 前端 (Vue 3 + TypeScript)
- **框架**: Vue 3.4+, Composition API
- **状态管理**: Pinia 2.x
- **路由**: Vue Router 4.x
- **样式**: Tailwind CSS 3.x (自定义色板)
- **图表**: Chart.js 4.x + vue-chartjs
- **HTTP**: Axios 1.x (智能响应解包)

## 关键特性

### ✅ 架构统一
- **单一后端**: Spring Boot 为主后端 (端口 8080)
- **数据持久化**: H2 文件存储 (开发) / MySQL (生产)
- **Docker 一致**: docker-compose.yml 正确指向 Spring Boot

### ✅ 功能完整
- **认证系统**: 密码-only 登录 (admin123/user123), 注册禁用
- **AI 集成**: 完整的 `/api/ai/*` 端点, 流式聊天支持
- **核心功能**: 金价追踪、技术文章、实用工具全部保留
- **智能适配**: 前端自动处理 Spring Boot 响应格式

### ✅ 部署就绪
- **开发模式**: `mvn spring-boot:run` + `npm run dev`
- **生产构建**: `mvn package` + `npm run build`
- **Docker 部署**: `docker-compose up --build`

## 启动命令

```bash
# 开发模式
cd backend && mvn spring-boot:run     # 终端1：后端
cd frontend && npm run dev            # 终端2：前端

# 一键启动 (Windows)
start.bat

# 生产部署
cd backend && mvn clean package -DskipTests && java -jar target/*.jar
```
