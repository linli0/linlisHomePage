# 壳层 + 首页 UI（方案 A）设计

**日期:** 2026-07-19  
**状态:** 已确认  
**范围:** `NavigationBar` / `FooterBar` / `HomeView` + 必要的 `main.css` 工具类

## 目标

暖色「咖啡台」视觉：琥珀 brand + 墨色 ink + 奶油底。第一屏与导航/页脚对比清晰、层次清楚，不改内页业务逻辑。

## 导航

- 首页且未滚动：半透明深底 `bg-ink-950/40` + 白字链接
- 滚动后或非首页：白底 / 深色模式 ink 底 + 墨色字 + 底边
- Logo：琥珀实色块 + 品牌名随状态变色
- 去掉无效 `accent-*`；链接触控区统一 `px-3 py-2 rounded-lg`

## 首页 Hero

- 满幅琥珀线性渐变（内联 style 保底）
- 主标题 Coffee / Cookies 白字
- 一句完整 slogan（白/白透明）
- CTA：主=白底琥珀字；次=白描边半透明底
- 去掉抢戏的 ping 绿点徽章（或改为静态细标签）

## 首页下文

- 功能区 / 金价区：奶油或白底，`brand`/`ink` 色板
- 金价文案：国内金价、黄金9999·元/克
- 清理无效 `gold-*` / `surface-*` / `accent-*` 类名

## 页脚

- 浅底、细顶边、与导航同品牌；链接与版权小字清晰

## 非目标

- 不改金价/Wiki/工具等内页布局
- 不引入新字体包或插画资源
