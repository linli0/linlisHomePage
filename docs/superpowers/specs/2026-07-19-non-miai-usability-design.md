# 非小爱主站可用性（通宵切片）

**日期:** 2026-07-19  
**状态:** 执行中（用户授权：睡眠中自主推进，中途不问）  
**范围:** 金价 / 首页 / AI / 工具 / 文章 / 登录设置 / 文档运维  
**排除:** 小米音箱、对话总线、完成播报（已合入，本切片不动）

## 目标

让「每天会点开的主站能力」更稳、更清楚，而不是堆新模块。个人站：少配置、少死链、错误可懂。

## 产品决策（锁定）

1. **默认栈叙事**：只宣传 FastAPI `:8000` + Vite `:3000` / Docker `api+frontend`；Spring 仅 legacy。
2. **金价**：国内 AU9999 元/克；手动刷新 + 后端定时；失败时保留上次成功价并明示。
3. **AI**：Ollama 优先；状态条要诚实（离线/模型列表空）；不默认烧 DeepSeek。
4. **工具箱**：纯前端/轻 API，失败给可读错误，不假装成功。
5. **文章**：公开列表空态友好；管理路径需登录（现状保持）。
6. **不做**：社交推文大改、量化策略重做、新 UI 框架、裸域 DNS（需 Cloudflare 手工）。
